本程序基于 JDK 8。

一、大部分的包所包含的内容都是一目了然的，下面仅对部分包做一下说明。

study.javase.awt.dd
    Drag and Drop（https://docs.oracle.com/javase/8/docs/technotes/guides/dragndrop/index.html）

study.javase.awt.graphic2d
    Java 2D API

study.javase.awt.imf
    Java Input Method Framework（https://docs.oracle.com/javase/8/docs/technotes/guides/imf/index.html）

study.javase.collection
    Java 集合，包括普通集合和并发集合

study.javase.collection.lagacy
    Java 早期的集合类（Vector、Hashtable 等）

study.javase.deployment
    与应用部署相关的内容，目前仅包括 Web Start 应用的部署，不包括 Applet 相关内容

study.javase.deployment.jnlp
    jnlp api: https://docs.oracle.com/javase/8/docs/jre/api/javaws/jnlp/index.html

study.javase.ext_mechanism
    Java 扩展技术（Extension Mechanism）。

study.javase.fullscreen_exclusive
    全屏独占模式（full-screen exclusive mode）的使用（https://docs.oracle.com/javase/tutorial/extra/fullscreen/index.html）

study.javase.functional.lambda
    lambda 表达式和函数接口（java.util.function)、方法引用的使用

study.javase.functional.stream
    java.util.stream.Stream 的使用

study.javase.jpda
    Java Platform Debugger Architecture（https://docs.oracle.com/javase/8/docs/technotes/guides/jpda/index.html）

study.javase.jvmti
    JVM Tool Interface（https://docs.oracle.com/javase/8/docs/technotes/guides/jvmti/index.html）

study.javase.langbasic
    与 Java 语言相关的基本概念，以及常用的类和接口的使用

study.javase.math
    java.math 包的使用

study.javase.net.sdp
    Socket Direct Protocol 的使用（InfiniBand）

study.javase.print
    javax.print 包的使用

study.javase.references
    java.lang.ref 包的使用

study.javase.security.policy
    Java 安全策略的配置和使用

study.javase.time
    Java 时间和日期 API 的使用（java.time 包)

study.javase.time.lagacy
    Java 早期的时间和日期 API（Date、Calendar）

study.javase.tools
    Java 常用工具的使用（https://docs.oracle.com/javase/8/docs/technotes/tools/index.html）

study.javase.tools.attach
    Dynamic Attach（https://docs.oracle.com/javase/8/docs/technotes/guides/attach/index.html）

study.javase.tools.jconsole
    JConsole Accessing API（https://docs.oracle.com/javase/8/docs/technotes/guides/management/jconsole.html）

study.javase.util
    java.util 包中除集合、时间日期类之外的其他工具类。

二、辅助类

study.javase.assist 包中的类是辅助类。

三、参考实现

study.javase.refimpl 中的子包是各个类的参考实现。
